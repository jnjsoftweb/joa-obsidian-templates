# 20-areas

**PARA 방법론**의 Areas 영역입니다. 완료 시점이 없는 지속적 관리 영역을 담습니다.

## 용도

- **책임 영역 노트**: 건강, 재무, 커리어, 관계 등 삶의 각 영역
- **루틴 & 프로세스**: 반복 업무 절차, 체크리스트
- **현황 파악**: 각 영역의 현재 상태와 목표

## 사용법

1. 영역별 폴더 또는 단일 노트로 관리
2. 정기적으로 각 영역의 상태를 점검하고 업데이트
3. 특정 영역에서 프로젝트가 생기면 `10-projects`에 생성하고 연결

## 영역 예시

- 건강 / 운동 / 식단
- 재무 / 투자
- 커리어 / 학습
- 인간관계 / 가족
