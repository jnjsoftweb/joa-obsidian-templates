# 91-attachments

노트에 첨부되는 이미지, PDF, 동영상 등 미디어 파일을 저장합니다.

## 용도

- **이미지**: 스크린샷, 사진, 다이어그램
- **문서**: PDF, Word, Excel 등 첨부 파일
- **기타 미디어**: 동영상, 오디오 클립

## 설정 방법

Obsidian 설정 → Files and Links → Default location for new attachments:
```
91-attachments
```

## 정리 팁

파일이 많아지면 연도/월 또는 주제별 서브폴더로 분류:
```
91-attachments/
  2026/
    03/
  screenshots/
  diagrams/
```
