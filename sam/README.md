# sam 볼트

Moon Jung Sam의 개인 및 가족 생활 관리를 위한 Obsidian 볼트입니다.

## 용도

- **인간 관계**: 친구, 친척, 지인 정보 및 교류 기록
- **신상 정보**: 가족 구성원 관련 중요 정보
- **계정 관리**: 계좌, 웹사이트, API 등 아이디/비밀번호 (암호화 플러그인 권장)
- **일상 생활**: 루틴, 건강, 취미 기록
- **만남 & 여행**: 약속, 여행 계획 및 후기

## 폴더 구조

| 폴더 | 용도 |
|------|------|
| `00-dashboard` | 홈 화면 및 자주 쓰는 노트 모음 |
| `01-inbox` | 빠른 메모 임시 저장소 |
| `02-daily` | 일일/주간 노트 |
| `10-projects` | 진행 중인 개인 프로젝트 |
| `20-areas` | 지속 관리 영역 (건강, 재무, 가족 등) |
| `30-resources` | 참고 자료 및 지식 |
| `40-archives` | 완료·종료된 기록 |
| `90-meta` | 템플릿, 설정 문서 |
| `91-attachments` | 이미지, 파일 첨부 |

## 보안 주의사항

계정 정보 노트는 **Obsidian Meld Encrypt** 또는 **Local Encryption** 플러그인으로 암호화하여 관리하세요.
