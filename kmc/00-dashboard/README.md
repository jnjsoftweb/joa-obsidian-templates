# 00-dashboard

볼트의 시작점이자 네비게이션 허브입니다.

## 용도

- **홈 노트**: 볼트 전체를 조망하는 메인 대시보드 노트
- **MOC (Map of Content)**: 주제별 노트 지도 — 관련 노트를 한 곳에서 탐색
- **고정 링크 모음**: 자주 방문하는 노트로의 빠른 접근

## 사용법

1. `Home.md` 또는 `Dashboard.md`를 이 폴더에 생성하고 볼트 시작 노트로 설정
2. Dataview 쿼리로 최근 수정 노트, 진행 중 프로젝트, 오늘 할 일 등을 동적으로 표시
3. 주제별 MOC 노트를 만들어 `30-resources` 노트들을 카테고리로 연결

## 권장 노트 예시

- `Home.md` — 메인 대시보드
- `MOC-projects.md` — 프로젝트 지도
- `MOC-knowledge.md` — 지식 지도
