# kmc 볼트

한의원 개원·진료·경영과 한의학 자료 관리를 위한 Obsidian 볼트입니다.

## 용도

- **한의원 개원**: 개원 관련 절차, 인허가, 시설 정보
- **진료 관리**: 진료 프로토콜, 처방 기록, 치료 케이스
- **경영 관리**: 매출/지출, 인사, 마케팅, 운영 이슈
- **한의학 자료**: 고전 문서, 용어집, 학술 자료 아카이브
- **앱 개발/운영**: 진료·경영 지원 앱 개발 및 운영 기록

## 폴더 구조

| 폴더 | 용도 |
|------|------|
| `00-dashboard` | 홈 화면 — 주요 지표, 오늘 진료 현황 |
| `01-inbox` | 진료 중 빠른 메모, 미분류 아이디어 |
| `02-daily` | 일일 진료 일지 및 업무 로그 |
| `10-projects` | 진행 중 프로젝트 (앱 개발, 개원 준비 등) |
| `20-areas` | 지속 관리 영역 (진료, 경영, 한의학 공부 등) |
| `30-resources` | 한의학 자료, 고전 문헌, 참고 논문 |
| `40-archives` | 완료된 프로젝트, 과거 케이스 |
| `90-meta` | 템플릿 (진료 노트, 처방 양식 등), 설정 |
| `91-attachments` | 이미지, 검사 결과, 스캔 문서 |

## 권장 플러그인

- **Dataview**: 진료 케이스 통계, 처방 빈도 분석
- **Kanban**: 진료 대기/처치/완료 흐름 관리
- **Calendar**: 일일 진료 일지 네비게이션
