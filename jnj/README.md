# jnj 볼트

jnjsoft(일인 소프트웨어 개인사업)의 개발 업무 및 사업 경영 관리를 위한 Obsidian 볼트입니다.

## 용도

- **외주 개발**: 클라이언트 프로젝트 요구사항, 진행 현황, 납품 기록
- **내부 개발**: 자체 제품/서비스 개발 기록
- **회계**: 매출, 지출, 세금계산서 관리
- **세무**: 부가세, 종합소득세 신고 관련 자료

## 폴더 구조

| 폴더 | 용도 |
|------|------|
| `00-dashboard` | 홈 화면 — 진행 프로젝트 현황, 마감 일정 |
| `01-inbox` | 클라이언트 요청, 아이디어 빠른 메모 |
| `02-daily` | 개발 일지, 작업 로그 |
| `10-projects` | 진행 중 외주/내부 프로젝트 |
| `20-areas` | 지속 관리 영역 (회계, 세무, 기술 역량 등) |
| `30-resources` | 기술 문서, API 레퍼런스, 개발 참고 자료 |
| `40-archives` | 완료된 프로젝트, 과거 계약 자료 |
| `90-meta` | 템플릿 (프로젝트 계획서, 견적서 양식 등), 설정 |
| `91-attachments` | 계약서, 산출물, 스크린샷 |

## 권장 플러그인

- **Dataview**: 프로젝트 수익성, 작업 시간 통계
- **Kanban**: 외주 프로젝트 진행 단계 관리
- **Git**: 개발 노트와 코드 저장소 연동 기록
