# 01-inbox

생각, 메모, 링크를 즉시 저장하는 GTD 방식의 수집함입니다.

## 용도

- **빠른 캡처**: 생각날 때 즉시 적어두는 임시 메모
- **웹 클리핑**: 브라우저에서 가져온 기사, 링크
- **미분류 노트**: 어느 폴더에 넣을지 아직 결정하지 못한 노트

## 사용법

1. 일단 여기에 저장 — 완벽하지 않아도 괜찮음
2. **주 1회 이상** inbox를 비우기: 각 노트를 적절한 폴더로 이동하거나 삭제
3. 오래된 미처리 노트가 쌓이지 않도록 관리

## 원칙

> inbox는 최종 목적지가 아닙니다. 항상 비워두는 것을 목표로 하세요.
