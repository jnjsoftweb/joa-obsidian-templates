# 40-archives

**PARA 방법론**의 Archives 영역입니다. 완료된 프로젝트와 더 이상 활성화되지 않는 자료를 보관합니다.

## 용도

- **완료된 프로젝트**: `10-projects`에서 완료 후 이동
- **비활성 영역**: `20-areas`에서 더 이상 관리하지 않는 영역
- **구버전 자료**: 업데이트로 대체된 오래된 참고 자료

## 사용법

1. 삭제하지 말고 이곳으로 이동 — 나중에 참조 가능
2. 연도별 서브폴더로 정리 권장: `40-archives/2025/`
3. Dataview로 아카이브 검색 시 `FROM "40-archives"`로 쿼리

## 원칙

> 완료 = 삭제가 아닙니다. 아카이브는 미래의 나에게 보내는 참고자료입니다.
